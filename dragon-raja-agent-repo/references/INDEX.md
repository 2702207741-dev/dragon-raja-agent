# 龙族蒸馏数据索引

## 快速查询指南

### 按角色查对话
```bash
# 查楚子航所有对话
grep "楚子航" 龙族*.md | grep "DIA|GOLD" 

# 查路明非所有对话  
grep "路明非" 龙族*.md | grep "DIA|GOLD"

# 查诺诺所有对话
grep "诺诺\|陈墨瞳" 龙族*.md | grep "DIA|GOLD"
```

### 按世界观查设定
```bash
# 查言灵相关
grep "WORLD.*言灵" 龙族*.md

# 查血统相关
grep "WORLD.*血统\|WORLD.*爆血\|WORLD.*S级" 龙族*.md

# 查龙族相关
grep "WORLD.*龙王\|WORLD.*龙类\|WORLD.*初代种" 龙族*.md
```

### 查伏笔
```bash
# 查所有跨卷未回收的伏笔
grep "跨卷未回收" 龙族*.md

# 查特定伏笔
grep "FW.*黑王\|FW.*路鸣泽\|FW.*奥丁" 龙族*.md
```

### 查写作模板
```bash
# 查所有可复用模板
grep "^T[0-9]_" 龙族*.md
```

### 查黄金对话
```bash
# 查所有黄金对话
grep "^GOLD_" 龙族*.md
```
