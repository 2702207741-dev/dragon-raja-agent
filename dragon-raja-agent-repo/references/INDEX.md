# 龙族蒸馏数据索引

## 快速查询指南

### 按角色查对话
```bash
# 查楚子航所有对话
grep -E "楚子航" 龙族*.md | grep -E "DIA|GOLD"

# 查路明非所有对话  
grep -E "路明非" 龙族*.md | grep -E "DIA|GOLD"

# 查诺诺所有对话
grep -E "诺诺|陈墨瞳" 龙族*.md | grep -E "DIA|GOLD"
```

### 按世界观查设定
```bash
# 查言灵相关（兼容多种格式）
grep -E "(^|\|)WORLD.*言灵" 龙族*.md

# 查血统相关
grep -E "(^|\|)WORLD.*(血统|爆血|S级)" 龙族*.md

# 查龙族相关
grep -E "(^|\|)WORLD.*(龙王|龙类|初代种)" 龙族*.md
```

### 查伏笔（兼容缩进格式 + 方括号格式）
```bash
# 查所有FW_条目（三种格式：^FW_ / 缩进FW_ / [FW_）
grep -E "(^|\s)"?FW_|\[FW_" 龙族*.md

# 查跨卷未回收的伏笔
grep -E "跨卷未回收|CROSS_VOLUME" 龙族*.md

# 查已回收的伏笔
grep -E "已回收|RESOLVED_IN_VOLUME" 龙族*.md

# 查特定关键词伏笔
grep -E "FW.*(黑王|路鸣泽|奥丁)" 龙族*.md
grep -E "FW.*(绘梨衣|赫尔佐格|圣骸)" 龙族*.md
```

### 查写作模板
```bash
# 查所有可复用模板
grep -E "^T[0-9]+_" 龙族*.md
```

### 查黄金对话
```bash
# 查所有黄金对话
grep -E "^GOLD_" 龙族*.md
# 其中龙族Ⅳ的金句在文件内标注为 [G01]-[G17]，额外用：
grep -E "\[G[0-9]+\]" 龙族Ⅳ_奥丁之渊.md
```

## 格式说明
- 龙族前传、龙族Ⅰ、龙族Ⅱ的 FW_ 使用 `FW_XXX | ...` 格式（行首或缩进）
- 龙族Ⅲ 的 FW_ 使用 `FW_III_XX | ...` 格式
- 龙族Ⅳ 的 FW_ 使用 `[FW_IV_XX] ...` 格式
- 以上 `grep -E "(^|\s)"?FW_|\[FW_"` 命令已兼容全部三种格式
