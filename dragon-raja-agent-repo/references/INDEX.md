    17|### 按世界观查设定
    18|```bash
    19|# 查言灵相关（兼容多种格式）
    20|grep -E "(^|\|)WORLD.*言灵" 龙族*.md
    21|
    22|# 查血统相关
    23|grep -E "(^|\|)WORLD.*(血统|爆血|S级)" 龙族*.md
    24|
    25|# 查龙族相关
    26|grep -E "(^|\|)WORLD.*(龙王|龙类|初代种)" 龙族*.md
    27|```

    29|### 查伏笔（兼容缩进格式 + 方括号格式）
    30|```bash
    31|# 查所有FW_条目（三种格式：^FW_ / 缩进FW_ / [FW_）
    32|grep -E "(^|\s)"?FW_|\[FW_" 龙族*.md
    33|
    34|# 查跨卷未回收的伏笔
    35|grep -E "跨卷未回收|CROSS_VOLUME" 龙族*.md
    36|
    37|# 查已回收的伏笔
    38|grep -E "已回收|RESOLVED_IN_VOLUME" 龙族*.md
    39|
    40|# 查特定关键词伏笔
    41|grep -E "FW.*(黑王|路鸣泽|奥丁)" 龙族*.md
    42|grep -E "FW.*(绘梨衣|赫尔佐格|圣骸)" 龙族*.md
    43|```

    52|```bash
    53|# 查所有黄金对话
    54|grep -E "^GOLD_" 龙族*.md
    55|# 其中龙族Ⅳ的金句在文件内标注为 [G01]-[G17]，额外用：
    56|grep -E "\[G[0-9]+\]" 龙族Ⅳ_奥丁之渊.md
    57|```

    59|## 格式说明
    60|- 龙族前传、龙族Ⅰ、龙族Ⅱ的 FW_ 使用 `FW_XXX | ...` 格式（行首或缩进）
    61|- 龙族Ⅲ 的 FW_ 使用 `FW_III_XX | ...` 格式
    62|- 龙族Ⅳ 的 FW_ 使用 `[FW_IV_XX] ...` 格式
    63|- 以上 `grep -E "(^|\s)"?FW_|\[FW_"` 命令已兼容全部三种格式
